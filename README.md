# WP API CodeIA

Plugin de WordPress para integración con la API de CodeIA.

## Descripción

Este plugin permite la integración de WordPress con la API de CodeIA.

## Requisitos

- WordPress 5.0 o superior
- PHP 7.4 o superior

## Instalación

1. Sube el plugin al directorio `/wp-content/plugins/`
2. Activa el plugin desde el menú 'Plugins' en WordPress
